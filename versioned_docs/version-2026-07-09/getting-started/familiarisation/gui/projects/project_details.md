# Project details

This guide covers the navigation and usage of the project details screen.

## Understanding the project details screen

Once you've selected a project, you'll be taken to its details screen, which includes several features:

### Features of the project details screen

<p align='center'>
  <img alt='Project details screen' src={require('!url-loader!../images/project-details.png').default} className='image-border'/>
</p>

This screen manages and defines the available agents, assets, asset templates, environments, Git remotes, properties, settings and workflows. By default, it shows the [project's activities table](/getting-started/familiarisation/gui/projects/activities.md#project-activities).

#### Buttons & links

| Buttons & links        | Function                                                               |
|------------------------|------------------------------------------------------------------------|
| **Pencil icon**        | Allows you to edit the details of the current project.                 |
| **Bookmark icon**      | Allows you add and remove bookmarks for the project for quick access.  |
| **Bell icon**          | Allows you to manage the project's notification settings.              |
| **Three dots icon**    | Allows you to archive the project.                                     |

These buttons and links are available in every project, environment and asset details screens and are used to manage the respective entity.
