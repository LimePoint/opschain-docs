# Overview

Upon [logging in](/getting-started/familiarisation/gui/login.md), you are first taken to the activity screen, which lists all the changes and workflow runs that are available to you. The left pane, used for navigation, lists the operations you can perform. The top pane allows you to log out, as well as condense the navigation bar thereby gaining more screen real estate.

The table below lists the key areas available in this screen.

| Key | Element             | Description                                                      |
|-----|---------------------|------------------------------------------------------------------|
| A   | Burger icon         | Condense/expand the navigation bar.                                  |
| B   | Search resource     | Global search for projects, environments, agents, assets, asset templates and workflows. |
| C   | Run activity        | Run a change or a workflow.                                      |
| D   | User preferences, secret tool and theme selection       | Logged in user notification preferences, the secret tool and theme selection (light and dark themes are available). The secret tool has a **Secret vault** tab to store or retrieve secret values and files in the secret vault — a stored value can be entered, uploaded as a file, or auto-generated by the vault — and an **AES encryption** tab to encrypt values without touching the vault.                      |
| E   | List filters        | Allows you to filter the contents of the list by various criteria. When filters are applied, the URL is updated to reflect the filters selected, making it easy to share the current view with others. The filters are persisted locally in the browser, so they will be remembered between sessions. |
| F   | List interface      | Each row in the list represents a change or workflow run matching the current filters and available to the user. The columns displayed can be customised. |
| G   | Administrative navigation bar | Provides access to OpsChain administrative functions and policy management. The administration menu is available only for users with administrative privileges. |
| H   | Manage activity     | Manage activity waiting for the current user's approval or continuation. The total count of activities waiting is displayed beside the text.            |

The top pane also shows an **active runs** indicator, displaying the number of changes and workflow runs that are currently in flight across all projects, environments and assets you can access. Selecting it opens a list of those runs, each linking through to its [activity details](/getting-started/familiarisation/gui/activity_details.md).
